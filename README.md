# ExpenseFlow UI

Super-modern multi-page expense + balance tracker frontend with a cinematic dark UI.

## Highlights

- Main page focuses on a large transaction entry surface (expense/income).
- Tiny icon-only side navigation for quick page switching.
- Settings-first flow: add accounts/cards before tracking.
- Category dropdown with popular categories (food, groceries, electronics, gadgets, etc.).
- Insights page: category breakdown, account balances, CC utilisation, monthly trend.
- History page: full table, filters, JSON backup export.
- Ambient cursor glow + 3D tilt interaction + animated glassmorphism cards.
- Dark/light theme toggle with persistence.

## Run

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

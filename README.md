# ExpenseFlow UI

A modern multi-page expense and balance tracker with:

- Main page for adding income/expense transactions (with account/card, debit/credit mode, custom date).
- Insights page with category breakdown, account balances, credit card utilisation, and monthly trend.
- History page with full transaction table, filters, and JSON export.
- Settings page to add accounts, credit cards, opening balances, currency, and monthly budget.
- Dark mode/light mode toggle, glassmorphism UI, and smooth animations.

## Run locally

You can directly open `index.html` in a browser, or run:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
